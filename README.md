# Python Scripts Collection

This repository contains various Python scripts for system monitoring, file operations, and website checking tasks. Each script performs a specific function that can be useful for automating common tasks on a system.

## Scripts Overview

### 1. `countingfiles.py`
This script finds all `.txt` files in a specified directory. It can be used to quickly search for text files and count how many are present.

### 2. `diskusage.py`
This script checks the percentage of used disk space on your system. It helps in monitoring disk usage to avoid running out of space.

### 3. `hostname.py`
This script retrieves the system's hostname and saves the output to a .txt file. It's useful for logging and system identification.

### 4. `readwritefile.py`
This script reads a .txt file, counts the number of lines, words, and characters, and then saves this information to another text file. It is useful for analyzing the contents of text files.

### 5. `website_checker.py`
This script checks a website's status and automatically retrieves the HTTP mail status. It helps in monitoring the health of a website.
